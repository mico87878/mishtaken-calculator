## כיצד לעדכן נתוני עיר

פתח את `data/cities.json` ומלא את השדות הריקים (`null`) לכל עיר:

```json
{
  "id": 5,
  "name": "חדרה",
  "units": 268,
  "lotteries": 4,
  "pricePerSqm": 12000,        ← מחיר למ"ר לפי גיליון ההגרלה (ללא מע"מ)
  "discountType": "B",         ← "A" = 25%/600K  |  "B" = 20%/300K
  "marketPricePerSqm": 22000,  ← שווי שוק אמיתי למ"ר (לפי עסקאות/שמאי)
  "availableRooms": [3, 4, 5], ← אילו גדלי דירה זמינים בהגרלה זו
  "notes": ""                  ← הערות חופשיות (לא מוצג למשתמש)
}
```

### ערכי discountType:
| ערך | משמעות |
|-----|--------|
| `"A"` | הנחה של **25%** או **600,000₪** — הנמוך מביניהם |
| `"B"` | הנחה של **20%** או **300,000₪** — הנמוך מביניהם |
| `null` | נתון עדיין לא ידוע — העיר תופיע עם תג "בקרוב" |

## לוגיקת חישוב — סיכום מלא

### שלב 1 — גודל דירה
| חדרים | גודל |
|-------|------|
| 3 חד' | 80 מ"ר |
| 4 חד' | 100 מ"ר |
| 5 חד' | 125 מ"ר |

### שלב 2 — מחיר דירה
```
מחיר למ"ר כולל מע"מ  = pricePerSqm × 1.18
מחיר לפני הנחה       = מחיר למ"ר כולל מע"מ × גודל

הנחה סוג A = min(מחיר לפני הנחה × 0.25 , 600,000)
הנחה סוג B = min(מחיר לפני הנחה × 0.20 , 300,000)

מחיר סופי = מחיר לפני הנחה − הנחה
```

### שלב 3 — כללי משכנתה
```
אם מחיר סופי ≤ 1,750,000:
  משכנתה     = 90% × מחיר סופי
  הון עצמי   = max(100,000 , מחיר סופי − משכנתה)
  משכנתה     = מחיר סופי − הון עצמי   ← עדכון אם הון עצמי = 100K

אם 1,750,000 < מחיר סופי ≤ 2,100,000:
  משכנתה     = min(1,575,000 , 75% × marketPricePerSqm × גודל)
  הון עצמי   = max(100,000 , מחיר סופי − משכנתה)

אם מחיר סופי > 2,100,000:
  משכנתה     = 75% × מחיר סופי
  הון עצמי   = 25% × מחיר סופי
```

### שלב 4 — החזר חודשי (שפיצר)
```
ריבית חודשית r = 4.7% / 12 = 0.003917
תקופה n        = 360 חודשים (30 שנה)
החזר חודשי     = משכנתה × (r × (1+r)^n) / ((1+r)^n − 1)
```

### שלב 5 — הכנסה נטו נדרשת
```
הכנסה נטו מינימלית = החזר חודשי ÷ 0.38
```
(ההחזר מהווה 38% מההכנסה הפנויה נטו)

### שלב 6 — רווח משוער
```
שווי שוק אמיתי = marketPricePerSqm × גודל
רווח משוער     = שווי שוק אמיתי − מחיר סופי
אחוז רווח      = רווח ÷ שווי שוק × 100
```

## מחשבון הצמדה למדד

### חוקים
- **20% ראשונים** מסכום העסקה — לא צמודים לעולם
- **80% הנותרים** — כל תשלום צמוד ל-50% משינוי מדד תשומות הבנייה
- **עוגן המדד** = תאריך החתימה על החוזה
- כשתשלום נפרע — ההצמדה מפסיקה

### לוח תשלומים סטנדרטי
| # | תאריך | אחוז |
|---|-------|------|
| 1 | חתימה | 7% |
| 2 | חתימה + 45 יום | 13% |
| 3-10 | כל 4 חודשים | 10% × 8 |

### חישוב הצמדה לכל תשלום
```
חלק לא צמוד = min(סכום תשלום , יתרת 20% הלא-צמודים)
חלק צמוד    = סכום תשלום − חלק לא צמוד

מקדם הצמדה  = 1 + (שינוי מדד מצטבר × 0.5)
תוספת מדד   = חלק צמוד × (מקדם הצמדה − 1)
סכום סופי    = חלק לא צמוד + חלק צמוד × מקדם הצמדה
```

### פרמטרים ניתנים לשינוי על ידי המשתמש
- **חודשי מסירה** (ברירת מחדל: 36)
- **צפי מדד שנתי** (ברירת מחדל: 2.5%)
